# Game Station

### Overview
Game Station is an e-commerce website that allows people to buy board games.

### Tech Stack
- Node
- Express
- React
- React Router
- Redux
- Axios
- JWT
- MongoDB
- React-Bootstrap

### Getting Started
1. Run `mongod`
2. Run  `npm run dev` in root directory of the project
