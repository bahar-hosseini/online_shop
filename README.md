### Getting Started
1. Run `mongod`
2. Run  `npm run dev` in root directory of the project

### Tech Stack
- Node
- Express
- React
- React Router
- Redux
- Axios
- JWT
- MongoDB
- React-Bootstrap


### Seed Database


#### Import data:

npm run data:import

#### Destroy data:
